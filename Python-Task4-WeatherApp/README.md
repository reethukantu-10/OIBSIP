# Basic Weather App

## Description

This project is a beginner-level Weather Application built in Python.

It fetches real-time weather information using the OpenWeatherMap API.

---

## Features

- Search weather by city name
- Displays:
  - Temperature in Celsius
  - Temperature in Fahrenheit
  - Humidity
  - Weather Condition
  - Wind Speed
- Handles:
  - Invalid city names
  - Empty input
  - Network timeout
  - Internet connection errors
- Allows multiple searches

---

## Technologies Used

- Python
- requests
- OpenWeatherMap API

---

## Install

```bash
pip install -r requirements.txt
```

---

## Run

```bash
python weather.py
```

---

## API

Create a free API key from:

https://openweathermap.org/api

Replace

```python
API_KEY="YOUR_API_KEY"
```

with your own key.
