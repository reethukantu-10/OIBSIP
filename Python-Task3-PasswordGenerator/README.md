# Random Password Generator

## Description

This is a beginner-level Random Password Generator developed in Python.

The program allows users to generate secure random passwords by choosing the desired length and character types.

---

## Features

- User chooses password length
- Minimum password length of 8 characters
- Include:
  - Uppercase Letters
  - Lowercase Letters
  - Numbers
  - Symbols
- At least two character types must be selected
- Generates a random password
- Input validation
- Generate multiple passwords without restarting

---

## Technologies Used

- Python
- random
- string

---

## Requirements

Python 3.x

No external libraries are required.

---

## Run

```bash
python password_generator.py
```
