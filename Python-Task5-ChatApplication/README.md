# Chat Application

## Description

A beginner-level real-time chat application built using Python sockets.

Two users can communicate over localhost.

---

## Features

- Server accepts multiple client connections
- Clients join with their names
- Real-time messaging
- Timestamp with every message
- User join notifications
- User disconnect notifications
- Runs on localhost

---

## Technologies

- Python
- socket
- threading

---

## Run

### Start the server

```bash
python server.py
```

### Open another terminal

Run

```bash
python client.py
```

Open another terminal again

```bash
python client.py
```

Now two users can chat.
