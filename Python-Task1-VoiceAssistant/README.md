# Voice Assistant

## Description

This is a beginner-level Python Voice Assistant developed as part of an internship project.

## Features

- Recognizes voice commands
- Greets the user
- Tells current time
- Tells current date
- Opens Google search
- Speaks responses using Text-to-Speech
- Handles voice recognition errors gracefully

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- datetime
- webbrowser

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Commands

- Hello
- What is the time?
- What is today's date?
- Search Python programming
- Bye
